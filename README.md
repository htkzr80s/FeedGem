FeedGem (Experimental)
概要
プログラム未経験の個人が、AIの全面的なサポートを受けて開発した実験的なRSSリーダーです。
1階層のみのシンプルなフォルダ管理と、SQLiteによる軽量な動作を特徴としています。

技術仕様
動作環境
OS: Windows (x64専用)

Runtime: .NET 10.0 (動作にはランタイムのインストールが必要です)

Framework: WPF

主な機能
フォルダによるフィード管理（1階層限定）

OPMLファイルのインポート / エクスポート

WebView2による記事表示

多言語対応（日本語 / 英語）
