FeedGem は、プログラム未経験の個人がAIによるコーディングのみで作成した、3ペイン型のシンプルなRSSリーダーです。
ポータブル形式、多言語対応やダークテーマなど、基本的な機能を備えています。

動作環境

    OS: Windows (x64専用)
    Framework: WPF
    必須コンポーネント: .NET 10.0 ランタイム

主な機能

    ポータブル型: インストール不要で動作可能
    多言語対応: 日本語/英語を標準搭載。JSONファイルの追加で他言語拡張も可能
    フォルダ管理: 1段階のフォルダによるフィード整理
    フィード探索: Webサイトからフィードを自動検出
    ダークテーマ: 目に優しい外観切り替え
    OPML対応: フィードのインポートおよびエクスポートに対応

現在の制限事項（短所）

    キーボードショートカットには対応していません。
    細かな詳細設定機能はありません。

使い方

	・下部のテキストボックスにURLを入力してRSSフィードを登録。
	・URLを探索して複数のフィードを見つけることもできます。
	・アプリ起動時、スリープ復帰時、1時間ごとに自動更新されます。
	・右下の歯車ボタンで設定メニューを開きます。
	・sample.en.jsonを編集し、Languageフォルダに入れることで言語設定のプルダウンに追加できます。
	・jsonファイルの_metadata内、"language" "locale"は必ず入力してください。
	・足りない翻訳個所は自動で英語で埋められます。
	・AIにsample.en.jsonを渡して"翻訳して"と頼むのが楽だと思います。


上級者向け機能

	・iniファイルを直接編集することで一部の設定を変更できます。
	・もし変更して動作がおかしくなった場合は、アプリを完全に終了してiniファイルを削除してください。
	
	データベースに保存するフィードごとの記事最大件数
	MaxArticleCount=30
	
	非暗号化通信（http）を許可するかどうか
	AllowInsecureHttp=False
